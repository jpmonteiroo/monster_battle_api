To get a local copy up and running, follow these simple example steps.

### Prerequisites

* Ruby (version 3.3.0)

### Installation

1.  **Clone the repo**:
    ```bash

    ```

    ```bash

    ```

    ```bash
    rails db:create db:migrate db:seed
    ```
4.  **Running the Application**:
    ```bash
    rails server
    ```
